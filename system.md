# System Rules

이 문서는 에이전트 실행 규칙을 정의합니다.

## 사이트 접근 기록
- 사용자가 "특정 사이트에 접속해줘" 또는 이에 준하는 요청을 하면, 아래 형식으로 MD 파일을 생성한다.
  - 파일명: `{site}-{YYYYMMDD}.md` (예: `daum.net-20260131.md`)
  - 내용: 접속 목적, URL, 실행 시간, 이후 수행 단계 기록
- 해당 사이트를 다시 호출할 때는 위 MD를 먼저 읽고 그 지침을 따른다.

## 명령 기록
- 이 세션에서 실행한 모든 MCP 명령은 일자별 로그에 기록한다.
  - 파일명: `command-log-{YYYYMMDD}.md`
  - 기록 항목: 시간, 명령명, 인자, 결과(요약)
