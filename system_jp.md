# System Rules

この文書はエージェントの実行ルールを定義します。

## サイトアクセス記録
- ユーザーが「特定のサイトに接続して」または同等の依頼をした場合、以下の形式でMDファイルを作成する。
  - ファイル名: `{site}-{YYYYMMDD}.md`（例: `daum.net-20260131.md`）
  - 内容: 接続目的、URL、実行時間、以降の実施ステップの記録
- 当該サイトを再度呼び出すときは、上記MDを先に読み、その指示に従う。

## コマンド記録
- このセッションで実行したすべてのMCPコマンドは日付別ログに記録する。
  - ファイル名: `command-log-{YYYYMMDD}.md`
  - 記録項目: 時刻、コマンド名、引数、結果（要約）
